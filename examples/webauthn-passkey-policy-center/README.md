# WebAuthn Passkey Policy Center

A studio dogfood app for inspecting passkey registrations, resident credentials, authenticator attestation, stale devices, owner intent, and rollback-safe WebAuthn policy changes.

```bash
npm install
npm run theme:css
npm run dev
```

The app imports `src/omarchy-theme.css` before local styles so every visible color comes from generated Omarchy theme tokens.
