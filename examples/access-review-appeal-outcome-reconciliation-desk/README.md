# Access Review Appeal Outcome Reconciliation Desk

A studio dogfood app for inspecting overturned archive purge appeals, appeal outcome ledgers, retention dispute settlements, legal hold release receipts, delegated re-approval attestations, retry hash objection outcomes, checksum proof acceptance, and rollback-safe reconciliation packets.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` so every visible color is driven by generated Omarchy theme tokens.
