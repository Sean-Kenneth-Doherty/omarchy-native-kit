# Emergency Access Runbook

A studio dogfood app for inspecting break-glass accounts, offline runbooks, contact handoffs, time-boxed access grants, and rollback-safe emergency access rehearsals.

```bash
npm install
npm run theme:css
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors tied to Omarchy theme variables.
