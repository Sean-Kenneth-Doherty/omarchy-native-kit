# Privileged Role Activation Planner

An Omarchy-native studio app for inspecting just-in-time privileged role activations, approval chains, risky assignments, standing access drift, owner intent, emergency elevation, and rollback-safe role deactivation plans.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors tied to generated Omarchy tokens.
