# Access Credential Cache Cleanup Desk

A studio dogfood app for inspecting stale credential caches, secret consumers, emergency lease leftovers, purge progress, owner receipts, failed cleanup jobs, service restart dependencies, and rollback-safe cleanup packets.

```bash
npm install
npm run theme:css
npm run dev
npm run build
```

The app imports `src/omarchy-theme.css` before local styles and uses Omarchy semantic tokens for visible colors.
