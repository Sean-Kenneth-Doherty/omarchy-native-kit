# Access Review Export Readiness Board

A studio dogfood app for inspecting access review export readiness, sealed packet coverage, missing evidence, reviewer blockers, delegated approvals, retry exceptions, audit destinations, and rollback-safe delivery trails.

```bash
npm install
npm run dev
npm run build
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors on Omarchy semantic tokens.
