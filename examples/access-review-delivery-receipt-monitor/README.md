# Access Review Delivery Receipt Monitor

A studio dogfood app for inspecting access review delivery receipts, audit destination acknowledgements, failed export retries, reviewer closeout notes, packet checksum drift, delegated owner escalations, and rollback-safe delivery evidence.

```bash
npm install
npm run dev
npm run build
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors on Omarchy semantic tokens.
