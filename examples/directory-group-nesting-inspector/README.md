# Directory Group Nesting Inspector

An Omarchy-native studio app for inspecting nested directory groups, transitive privilege, stale owners, circular membership, app entitlements, blast-radius previews, and rollback-safe group cleanup.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors tied to generated Omarchy tokens.
