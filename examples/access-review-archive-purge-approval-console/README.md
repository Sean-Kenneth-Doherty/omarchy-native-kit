# Access Review Archive Purge Approval Console

A studio dogfood app for inspecting access review archive purge approvals, retention policy exceptions, legal hold overrides, delegated owner approvals, retry hash preservation, checksum proof retention, and rollback-safe purge authorization packets.

```bash
npm install
npm run dev
npm run build
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors on Omarchy semantic tokens.
