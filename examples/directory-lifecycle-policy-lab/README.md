# Directory Lifecycle Policy Lab

An Omarchy-native studio app for inspecting joiner-mover-leaver lifecycle policy, stale identities, manager handoffs, suspended accounts, entitlement drift, exception windows, and rollback-safe lifecycle cleanup.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors tied to generated Omarchy tokens.
