# Access Review Receipt Escalation Desk

A studio dogfood app for inspecting late access review receipt escalations, delegated owner paging, audit lane breach timers, retry transcripts, checksum reconciliation, reviewer closeout evidence, and rollback-safe escalation packets.

```bash
npm install
npm run dev
npm run build
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors on Omarchy semantic tokens.
