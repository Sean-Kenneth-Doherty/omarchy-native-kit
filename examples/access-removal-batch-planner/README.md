# Access Removal Batch Planner

An Omarchy-native studio app for planning access cleanup batches with entitlement dependencies, dry-run outcomes, delegated owners, rollback commands, communication windows, and audit-ready evidence.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` so every visible color can come from generated Omarchy tokens.
