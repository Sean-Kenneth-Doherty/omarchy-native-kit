# Prompt Foundry

Prompt Foundry is an Omarchy-native studio for turning theme context, blueprint kind, and workflow notes into a practical app-building prompt.

```bash
npm install
npm run theme:css
npm run dev
```

It dogfoods the agent contract:

```bash
omarchy-native agent prompt
omarchy-native agent blueprint --app prompt-foundry --kind studio
omarchy-native verify .
```
