# Access Owner Escalation Map

An Omarchy-native studio app for inspecting access owner escalation paths, stalled approvals, delegated backups, breach notifications, team handoffs, escalation evidence, and rollback-safe accountability trails.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` so every visible color can come from generated Omarchy tokens.
