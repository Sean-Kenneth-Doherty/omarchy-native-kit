# Access Review Purge Thaw Authorization Console

A studio dogfood app for inspecting archive purge thaw authorization requests, freeze release attestations, legal hold thaw clearances, delegated owner resume approvals, retry hash thaw proofs, checksum acceptance unlocks, and rollback-safe purge resume packets.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` so every visible color is driven by generated Omarchy theme tokens.
