# Access Review Archive Retention Policy Lab

A studio dogfood app for inspecting access review archive retention policies, ledger expiry windows, receipt closeout holds, owner acknowledgement history, retry hash preservation, checksum repair retention, and rollback-safe archive purge previews.

```bash
npm install
npm run dev
npm run build
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors on Omarchy semantic tokens.
