# Access Removal Evidence Vault

An Omarchy-native studio app for inspecting completed removal batches, sealed entitlement snapshots, rollback expiry, communication receipts, auditor requests, exception reopenings, and cleanup evidence retention.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` so every visible color can come from generated Omarchy tokens.
