# Access Reopen Exception Console

An Omarchy-native studio app for inspecting access reopen requests, temporary regrant approvals, dependency evidence, expiry timers, owner acknowledgements, rollback scope, and audit-ready exception trails.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` so every visible color can come from generated Omarchy tokens.
