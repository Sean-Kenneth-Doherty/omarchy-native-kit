# Directory Owner Attestation Desk

An Omarchy-native studio app for inspecting directory resource owners, stale approvers, delegated stewardship, missing attestations, group risk, escalation windows, and rollback-safe owner reassignment.

```bash
npm install
npm run build
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors tied to generated Omarchy tokens.
