# API Token Scope Auditor

A studio dogfood app for inspecting API token scopes, stale grants, owner intent, overbroad permissions, revocation windows, and rollback-safe scope reductions.

```bash
npm install
npm run theme:css
npm run dev
```

The app imports `src/omarchy-theme.css` before `src/styles.css` and keeps visible colors tied to Omarchy theme variables.
